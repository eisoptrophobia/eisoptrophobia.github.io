# Mr. Bones
A Discord bot for tabletop RPG servers

Commands:
<ul>
  <li>*ping: Pings the bot</li>
  <li>*help: Gets help</li>
  <li>*roll: Rolls dice</li>
  <li>*vote: Creates a vote</li>
  <li>*char: Manages Smyth characters</li>
</ul>
