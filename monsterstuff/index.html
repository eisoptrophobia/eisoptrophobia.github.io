Guide

Write each ability on its own line, without any blank lines in between. Bold text should be in [[text]], italics in ::text::, and undelined text in __text__. Replace text in "quotes" with whatever makes sense there.

Action Modifiers

Write these in at the end of an action description

  • This attack gets +"value" to the "attack/damage" roll.
  • This action has a cooldown of "value" "round/rounds".
  • On a hit, "effect description".
